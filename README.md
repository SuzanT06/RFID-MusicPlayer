# RFID-MusicPlayer
This is modernized Music Player that uses Raspberry Pi 3B along with a MFRC522 RFID Scanner and Python to make a Music Player using SPotify's API Key

This projects uses a Rapberry Pi 3b with the Legacy Bookworm Rapberry OS and a MFRC522 as hardwares. Additionally it uses python to fetch the spotify api data for the songs and connects the hardware and software aspects together.
